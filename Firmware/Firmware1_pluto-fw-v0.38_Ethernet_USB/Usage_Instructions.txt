1) Format the SD card to FAT32 format and copy the files from the SD card firmware into the SD card.

2) Insert the SD card into the SD card slot, set the dip switch to SD card boot mode, insert the Ethernet cable (or USB cable), and power on.

3) If using USB communication, you need to install the USB virtual network card driver in advance. Once installed, the official upper-level machine can be used in normal mode.

4) If using the Ethernet port, you need to set the IP address. Also, plug in the USB for the JTAG port. The default login credentials for the firmware are: username: root, password: analog. Then, configure the IP address using the command: ifconfig eth0 192.168.1.xx. Once done, the official upper-level machine can be used in normal mode.

5) Be mindful of the board's heat dissipation. For long-term operation, a CNC heat sink or fan is recommended. Thank you for your understanding!
